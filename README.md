# 棱镜视界 (Prism Vision)

这是一个使用 [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app) 引导的 [Next.js](https://nextjs.org) 项目。

## 项目简介

棱镜视界是深圳瓴克网络科技有限公司旗下的独立品牌。我们是一家聚焦文化传媒、游戏发行运营、创意软件及数字资产开发的综合性数字创意企业。

**核心理念：** 折射多元视角，创造无限世界。

我们致力于将前沿技术与艺术表达相融合，在以下四个维度持续产出高品质的数字体验：
- **游戏：** 探索互动的无限可能。
- **内容：** 打造沉浸式的视听盛宴。
- **工具：** 赋能创作者的数字利器。
- **资产：** 构建数字世界的基石。

## 快速入门

首先，运行开发服务器：

```bash
npm run dev
# 或
yarn dev
# 或
pnpm dev
# 或
bun dev
```

在浏览器中打开 [http://localhost:3000](http://localhost:3000) 查看结果。

你可以通过修改 `app/page.tsx` 开始编辑页面。当你修改文件时，页面会自动更新。

本项目使用 [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) 自动优化并加载 [Geist](https://vercel.com/font)（Vercel 的新字体家族）。
